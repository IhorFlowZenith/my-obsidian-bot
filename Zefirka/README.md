# Zefirka - Smart Daily Assistant

Welcome to Zefirka's data folder! This is where all bot data is stored and managed.

## 📁 Folder Structure

```
Zefirka/
├── SKILLS.md              # Bot skills and rules
├── README.md              # This file
├── user_profile.json      # User settings and preferences
├── tasks.json             # All tasks and categories
├── finances.json          # Expenses and budget
├── projects.json          # Pet projects and ideas
├── reminders.json         # Reminder settings
├── daily_plans.json       # Daily plans and history
├── daily_notes/           # Daily summaries (auto-generated)
│   ├── 2026-01-15.md
│   ├── 2026-01-16.md
│   └── ...
└── archive/               # Old data (auto-archived)
    ├── tasks_archive.json
    ├── finances_archive.json
    └── ...
```

## 🤖 What is Zefirka?

Zefirka is your smart daily assistant that helps you:
- 📋 Manage tasks and priorities
- 💰 Track expenses and budget
- 🚀 Manage pet projects
- 🧠 Get intelligent advice
- 📊 Analyze your productivity

## 🚀 Getting Started

1. **Read SKILLS.md** - Understand what Zefirka can do
2. **Update user_profile.json** - Set your preferences
3. **Start using** - Send messages to the bot on Telegram

## 📊 Data Files

### user_profile.json
- Your preferences
- Language, timezone, budget
- Statistics

### tasks.json
- All your tasks
- Priority levels
- Categories

### finances.json
- Expenses by category
- Monthly budget
- Spending history

### projects.json
- Your pet projects
- Project ideas
- Progress tracking

### reminders.json
- Reminder settings
- Daily reminder times
- Smart reminder rules

### daily_plans.json
- Daily task plans
- Completion tracking
- Historical data

## 🔄 Auto-Generated Files

### daily_notes/
Created automatically each day with:
- Tasks completed
- Expenses
- Mood/reflection
- Summary

### archive/
Old data is automatically archived:
- Tasks older than 3 months
- Expenses older than 6 months
- Completed projects

## ⚙️ Settings

Edit `user_profile.json` to customize:
- Language preference
- Reminder times
- Monthly budget
- Timezone

## 🔐 Privacy

- All data is stored locally in Obsidian
- Synced to GitHub (private repo)
- Never shared with third parties
- You control what's stored

## 📞 Support

If something doesn't work:
1. Check SKILLS.md
2. Verify JSON files are valid
3. Check bot logs
4. Restart the bot

## 🎯 Tips

- Be specific with commands
- Use natural language
- Provide context when needed
- Review daily summaries
